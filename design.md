# CV Web App Design Style Guide

## Design Philosophy

### Color Palette
- **Primary**: Deep Navy (#0B1426) - Professional, trustworthy, tech-focused
- **Secondary**: Electric Teal (#00D4AA) - Modern, innovative, AI-inspired
- **Accent**: Warm Gold (#FFB800) - Achievement, excellence, recognition
- **Background**: Charcoal (#1A1A1A) - Sophisticated, modern, reduces eye strain
- **Text**: Light Gray (#F8F9FA) - High contrast, readable
- **Muted**: Slate Gray (#64748B) - Supporting text, subtle elements

### Typography
- **Display Font**: "Canela" - Bold, editorial serif for headings and hero text
- **Body Font**: "Suisse International" - Clean, modern sans-serif for content
- **Code Font**: "JetBrains Mono" - Technical content and code snippets
- **Hierarchy**: Large display headings (48px+), medium section headers (32px), body text (16px)

### Visual Language
- **Aesthetic**: Modern tech editorial - combining professional sophistication with cutting-edge innovation
- **Mood**: Confident, intelligent, forward-thinking, approachable expert
- **Style**: Clean minimalism with strategic use of advanced visual effects
- **Inspiration**: Tech conference presentations, AI research publications, modern SaaS platforms

## Visual Effects & Styling

### Used Libraries
- **Anime.js**: Smooth micro-interactions and element animations
- **ECharts.js**: Interactive data visualizations for skills and project metrics
- **p5.js**: Creative coding for background particle effects and neural network visualizations
- **Pixi.js**: High-performance visual effects for hero section
- **Typed.js**: Typewriter effect for dynamic text in hero section
- **Splide.js**: Smooth carousel for project showcase
- **Shader-park**: Advanced background shaders for depth and movement

### Animation & Effects
- **Hero Background**: Subtle particle system representing neural networks with floating nodes
- **Text Effects**: Typewriter animation for name and title, color cycling emphasis on key skills
- **Scroll Motion**: Gentle reveal animations (16-24px translation, 150-300ms duration)
- **Hover Effects**: 3D tilt on cards, glow effects on interactive elements, smooth color transitions
- **Data Visualization**: Animated radar charts, progress bars, and metric counters

### Header & Navigation
- **Style**: Fixed transparent header with blur backdrop
- **Effect**: Subtle shadow on scroll, smooth color transitions
- **Layout**: Logo left, navigation center, contact button right
- **Mobile**: Hamburger menu with smooth slide-in animation

### Layout Principles
- **Grid System**: 12-column responsive grid with consistent spacing
- **Spacing**: 8px base unit system (8, 16, 24, 32, 48, 64px)
- **Sections**: Clear visual hierarchy with consistent padding and margins
- **Content Width**: Maximum 1200px centered with proper side margins
- **Responsive**: Mobile-first design with breakpoints at 768px, 1024px, 1440px

### Interactive Elements
- **Buttons**: Rounded corners, gradient backgrounds, hover lift effects
- **Cards**: Subtle shadows, hover transformations, smooth transitions
- **Forms**: Clean inputs with focus states and validation feedback
- **Charts**: Interactive tooltips, smooth animations, responsive design

### Background & Atmosphere
- **Primary Background**: Consistent dark gradient from charcoal to deep navy
- **Decorative Elements**: Subtle geometric patterns, circuit board motifs
- **Visual Depth**: Layered elements with proper z-index and shadow hierarchy
- **Consistency**: Same background treatment across all pages and sections

This design approach creates a professional, modern web presence that reflects your expertise in machine learning while maintaining excellent usability and visual appeal.
